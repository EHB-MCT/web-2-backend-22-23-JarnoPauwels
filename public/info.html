Course Project api

  • GET / - Documentation

    • Playlists

  • GET /allplaylists - Get all Playlists (Testing Purposes)
  • GET /playlists?userId=# - Get Playlist From A User
  • GET /playlist?userId=#&playlistId=# - Get One Playlist From A User
  • DEL /playlist?userId=#&playlistId=# - Delete One Playlist From A User
  • DEL /playlist/song?userId=#&playlistId=#&songId=# - Delete Song by Id
  • POST /playlist/song?userId=#&playlistId=#&songId=# - Add Song To Playlist
  • PUT /playlist - Create Playlist

    • Users

  • GET /users - Get all Users (Testing Purposes)
  • POST /register - Register User
  • POST /login - Login User
